# 2サイト分離リニューアル計画

## 結論

この案件は、2サイトを1つの複合LPに統合せず、役割を分けて制作します。

- `sites/algo-official/`: `https://algo-algorithm.jp/` 向けの本体サイト下地
- `sites/arugo-lp/`: `https://arugo-lp.jp/` 向けの広告LP下地

## 役割

### algo-algorithm.jp

遺品整理・不用品買取・回収・室内清掃を中心に、料金表、FAQ、ギャラリー、ブログ、WEB予約導線を持つ本体サイトとして扱います。SEO、信頼獲得、比較検討に耐える情報量を担います。

### arugo-lp.jp

引っ越し・生前整理・不用品回収の問い合わせ獲得に絞る広告LPとして扱います。ブログや読み物は増やしすぎず、無料見積もり、電話、フォームへの導線を優先します。

## 公開前に必ず確定する情報

- 会社の正式所在地、郵便番号、事務所表記
- 代表者名
- メールアドレス
- 問い合わせフォームURL
- WEB予約URL
- 営業時間、定休日
- 対応エリア
- 決済手段
- キャンセル規定
- 写真の商用利用権、人物・現場の掲載許諾

## 納品判定

現時点の雛形は開発中です。各サイトの `npm run preflight` が失敗する限り、公開・納品不可です。
