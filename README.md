# あるご 遺品整理・不用品回収・引越し LP

株式会社あるご向けのランディングページ一式です。

## プレビュー

- [あるご-プレビュー.html](./あるご-プレビュー.html)
- GitHub Pages: <https://anetai-droid.github.io/arugo-ihinseiri-kaishu-hikkoshi/%E3%81%82%E3%82%8B%E3%81%94-%E3%83%97%E3%83%AC%E3%83%93%E3%83%A5%E3%83%BC.html>

ローカルで確認する場合は、`あるご-プレビュー.html` をブラウザで開いてください。

## 主なファイル

- `あるご-プレビュー.html`: 完成プレビュー
- `あるご-コンセプト.html`: コンセプトページ
- `あるご-ブログ.html`: ブログページ
- `あるご-サイトマップ.html`: サイトマップ
- `あるご-プライバシー.html`: プライバシーポリシー
- `あるご-法定表記.html`: 法定表記
- `starter/`: Vite 版の実装ファイル
- `standalone/`: 単体で開ける HTML 版
- `画像素材/`: LP 用画像素材

## 開発

```powershell
cd starter
npm install
npm run dev
```

納品前の確認:

```powershell
npm run build
npm run preflight
```
